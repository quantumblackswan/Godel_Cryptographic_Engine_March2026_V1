# Security Policy

## Scope

This repository is a public research prototype and reproducibility artifact.
It is not advertised as production-ready cryptographic software.

## Please report

Please report issues involving:

- incorrect security claims
- benchmark misreporting
- broken verification logic
- backend mis-selection
- packaging errors that could mislead readers

## Not in scope

The following are already known limitations:

- no full IND-CPA, IND-CCA, or UC proof of the complete construction
- no constant-time claim for the Python orchestration layer
- wrapper portability layer not fully validated across environments
